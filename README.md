📘 Student Management System

A full-stack School Management System built using the MERN stack that enables administrators and teachers to efficiently manage students, classes, attendance and subjects.

🚀 Features
👨‍💼 Admin
Dashboard with analytics (students, teachers, classes)
Manage:
Students
Teachers
Classes
Subjects
Approve / block teachers
Payment management (PayPal demo integration)
👨‍🏫 Teacher
Secure login (role-based)
Mark daily attendance
Edit attendance (controlled)
View attendance history
View attendance statistics (graph)


🔐 Authentication & Security
JWT-based authentication
Role-based access control (Admin / Teacher)
Protected routes
Refresh token system
Password hashing with bcrypt
🛠️ Tech Stack
Frontend
React + TypeScript
Tailwind CSS
Axios
Recharts (graphs)
Zod (validation)
React Router


Backend
Node.js
Express.js
MongoDB + Mongoose
JWT Authentication
bcryptjs


🧠 Key Highlights
✅ Scalable modular backend architecture
✅ Role-based authentication system
✅ Real-world workflow (admin approval system)
✅ Clean UI with responsive design
✅ Data visualization using charts
✅ Form validation using Zod
✅ Production-ready folder structure

PayPal integration is implemented for demo purposes only
No real transactions are processed
